Ace Grid

A simple 12 column responsive grid system

Download

 

Ace Grid is a 12-colunm grid that allows you to write better layouts for your website, that can be used on any device including mobile and tables and you will be confident that your website will always looks to the audience.

Ace Grid columns are placed in rows, which are contained within a container. The container is set to a maximum width of 64rem that is 1024px, you can also change this value is needed with in the CSS file or buy editing the SCSS file that gives more control of how the grid functions.
